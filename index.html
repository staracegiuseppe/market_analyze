🔑 API esterne — come configurarle
FRED (Macro USA) — Gratuito, no limiti pratici
1. Registrati su fred.stlouisfed.org
2. Copia la key → imposta fred_api_key nel config HA
Abilita: Fed Funds, CPI USA, Treasury 2Y/10Y, SOFR
EIA (Energia) — Gratuito
1. Registrati su eia.gov/opendata
2. Copia la key → imposta eia_api_key
Abilita: prezzi WTI/Brent, scorte petrolio
FMP — Financial Modeling Prep — $19-29/mese
1. Abbonati su financialmodelingprep.com
2. Copia la key → imposta fmp_api_key
Abilita: 13F filings, insider trading, fundamentals (P/E, revenue growth, FCF)
Senza FMP: analisi a 4 layer (tech + macro + regime + settore)
Con FMP: analisi a 6 layer + dati istituzionali 13F + fundamental sanity check
? Gmail OAuth2 — setup
1. console.cloud.google.com → Crea progetto → Abilita Gmail API
2. Credenziali → ID client OAuth → App Desktop → scarica JSON
3. Sul tuo PC: pip install google-auth-oauthlib
4. python3 -c "from google_auth_oauthlib.flow import InstalledAppFlow; ..."
5. Copia client_id, client_secret, refresh_token nel config addon HA
